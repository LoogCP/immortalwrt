### 小结

- **DNS 配置**：
  国内的 DNS 可以填写运营商提供的 DNS 地址。

- **DNS 和路由规则顺序**：
  1. **优先写拒绝规则**（如 `reject` 或 `block`）。
  2. **其次写小范围规则**（如 `telegram` 或 `alibaba`）。
  3. **然后写大范围规则**（如 `cn`）。
  4. **最后设置回退规则**：
     - 使用备用 DNS（如 Google DNS）。
     - 目的是防止 DNS 泄漏，同时避免回环问题。

通过以上顺序配置，能够有效避免 DNS 回环问题，并提高 DNS 查询效率。
